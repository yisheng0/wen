## 项目介绍

该项目技术选型：后端采用springboot + mybatis + mysql + jwt，前端采用vue + element-ui + pinia + axios，分为登录界面和文章管理界面

登入界面可进行注册登入，登入后跳转到文章管理界面
文章管理界面分为文章分类，文章管理，灵感创作，个人中心
